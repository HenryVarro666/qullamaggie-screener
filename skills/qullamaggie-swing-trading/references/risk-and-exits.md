# 风险、仓位、止盈与 Edge

## R 与盈亏比的数学
- **R = 单笔交易的止损金额**。账户 $10,000、单笔止损 1% → 1R = $100。一笔赚 3R/5R 即盈亏比 3:1 / 5:1。用 R（而非金额或百分比）沟通交易。
- **期望值 E > 0 才有可执行性**——这是交易系统的核心铁律。
  - `E = P_win × AvgWin − P_loss × AvgLoss`，或比例形式 `E = P_win × W − P_loss`（W = 盈亏比 = AvgWin/AvgLoss）。
  - 只要 **E > 0** 长期就盈利；若 **E ≤ 0**，资金管理（凯利公式等）再完美，终点也必然破产。
  - 示例：30% × 15% − 70% × 3% = 0.024 > 0；30% × 15% − 70% × 5% = 0.01 > 0。
- 顶级交易员（Qullamaggie、Martin Luk，年化 ~300%、峰值 ~1000%）胜率通常仅 **~25%**；USIC 前十平均胜率约 30%。**低胜率是常态**，靠高盈亏比取胜。
- 同样回报下，**止损越紧 → 盈亏比越高**，而且是**抛物线式放大**：把止损从 3% 收紧到 1.5%，
  - 25% 收益的票多赚 **+8.34R**；
  - 50% 收益多赚 **+16.67R**；
  - 75% 收益多赚 **+25R**。
  - 回报越大，收紧止损带来的 R 增量越夸张——这正是"紧止损"铁律的数学根据。

## 真实统计（信念的实证锚点）
黄哥/Qullamaggie 风格的真实战绩，印证"低胜率、高盈亏比、靠少数大赢家"：
- **2025**：209 胜 / 731 负，胜率 **22.23%**，均盈 **+14.12%** / 均亏 **−2.08%**，赢家平均持仓 **3.79 天**、输家 1.31 天。年回报约 **+969%**。
- **2024**：104 胜 / 319 负，胜率 **23.81%**，均盈 +14.76% / 均亏 −2.71%，赢家持仓 5.38 天。年回报约 +344%。
- 关键画像：**赢家拿得久（均盈是均亏的 ~6-7 倍）、输家砍得快（持仓 1-1.5 天）**。这就是"让利润奔跑、快速止损"的数字长相。
- 核心指标体系：**EMA 9/21/50** + **Dollar volume（成交额）** + **AVWAP**。（注：本 skill 各处的 EMA10/EMA20/EMA50 即指 9/21/50 这一组，按培训统一记为 **EMA 9/21/50**。）

## 仓位规则
- **单笔账户风险**：黄哥铁律 ≤ 0.5%；Qullamaggie 多为 0.3-0.5%，极少 > 1%。
- **单笔股价止损幅度**：≤ 5%，绝大多数 ≤ 3%；且止损宽度 ≤ 股票 ADR/ATR。
- **单仓位规模**：账户 5-25%，多数 10-15%，看流动性、把握度、风险性。
- **隔夜上限**：任何单只股票/ETF 隔夜 ≤ 30%（日内可更高，因无隔夜跳空风险）。
- **仓位反推公式**：`股数 = (账户金额 × 单笔风险%) ÷ (进场价 − 止损价)`。
  - 例：账户 $50,000、风险 0.5%（=$250）、进场 $20、止损 $19（每股险 $1）→ 250 股。

## 分批止盈细则
基础规则：进场后 **3-5 天卖 1/3~1/2 + 止损移到保本**，余仓 **EMA10（快）/EMA20/EMA50（慢）trailing**，**收盘价首次跌破均线**才卖；新手用 EMA10。
- "3 天卖 30% + 止损提到成本之上"= 这笔无论如何都赚钱 → 内心有安全感 → 拿得住 → 让利润奔跑。
- 黄哥的四段细化（可选）：
  1. 到 **3R** 卖 25%；
  2. **过度延伸**（距 EMA50 通常 > ~10×ATR/ADR，具体票具体看）卖 25%；
  3. 从**高点回落 1.5×ADR/ATR** 卖 25%；
  4. **收盘跌破 EMA10 或 EMA21** 卖最后 25%。
- 止损：市价止损（market stop），不用限价止损。临近止损可挂硬止损，不必盯盘。
- 超级大牛股（斜率 > 45°、一路不回头）：收盘价接近 EMA10 时不用等 base，次日日内找高盈亏比点位直接进（个人心得，谨慎使用，例：VSH）。

## 发现 Edge 的清单
最底层逻辑：现在是**买方强势还是卖方强势**——买方强做多有优势，卖方强做空有优势。
1. **市场环境**：QQQ/VOO 周线看趋势（牛/熊/回调）。
2. **紧（Tight）**：Inside bar、价格吸筹、波动收窄。
3. **枯竭（exhaustion）**。
4. **AVWAP**（锚定 VWAP）。
5. **大成交量且突破前高**：通常意味机构介入、盈利惊喜或重大催化剂。
6. **ADR ≥ 3%**：不交易 ADR < 3% 的票。
7. **相对强度 RS**：大盘回调/走弱时股价仍坚挺甚至创新高；找 RS 线在股价创新高前**先创新高**的票——市场压力释放后最可能成为领头羊。
8. **RSI 背离**（动能 vs 价格的冲突信号）：
   - **看空背离（更重要）**：价格**创新高**，但 RSI **没创新高** → 上涨动能在减弱 → 可能要回调（提示止盈/不追）。
   - **看多背离**：价格**创新低**，但 RSI **没创新低** → 下跌动能衰竭 → 可能反弹（但本方法不抄底，仅作观察）。
9. **个股的 Stage / 波浪**：只做周线 Stage 2（价在 30 周线上 + 30 周线上行；判定细节见 `screening.md`）。

## ADR / ATR 定义
- **ADR（Average Daily Range）**= 过去 20 个交易日的日均波动幅度（%）。TC2000 公式：`100×((H0/L0+...+H19/L19)/20 − 1)`。TradingView 用现成的 ADR 指标即可（见 tradingview-setup.md）。
- **ATR（Average True Range）**= 平均真实波幅（绝对价格，非百分比）。
- 趋势交易靠股票爆发力赚钱：承担同等风险（靠好 setup），用高 ADR 换更高收益。
